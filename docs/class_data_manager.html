QString DataManager::getDataFilePath ( const QString & baseFileName,
int id = -1 )
static

Generuje ścieżkę do pliku danych na podstawie nazwy i identyfikatora.

Tworzy pełną ścieżkę do pliku w katalogu danych aplikacji, uwzględniając nazwę bazową oraz opcjonalny identyfikator. Jeśli katalog nie istnieje, zostanie utworzony.

Parametry
baseFileNameBazowa nazwa pliku (np. "stations").
idIdentyfikator, domyślnie -1 (brak identyfikatora w nazwie pliku).
Zwraca
QString Ścieżka do pliku w formacie JSON.

Tworzy pełną ścieżkę do pliku w katalogu danych aplikacji, używając nazwy bazowej i opcjonalnego identyfikatora. Jeśli katalog nie istnieje, tworzy go. Nazwa pliku ma format "<baseFileName>.json" lub "<baseFileName>_<id>.json".

Parametry
baseFileNameBazowa nazwa pliku (np. "stations").
idIdentyfikator, domyślnie -1 (brak identyfikatora w nazwie pliku).
Zwraca
QString Ścieżka do pliku w formacie JSON.

Wczytuje wszystkie historyczne dane pomiarów dla danego identyfikatora.

Wyszukuje pliki z danymi historycznymi dla podanego typu ("measurements") i identyfikatora, wczytuje ich zawartość oraz datę utworzenia na podstawie nazwy pliku. Zwraca wektor par (czas, dane).

Parametry
typeTyp danych ("measurements").
idIdentyfikator czujnika, dla którego wczytywane są dane.
Zwraca
QVector<QPair<QDateTime, QByteArray>> Wektor par zawierających czas i dane.

Wyszukuje pliki z danymi historycznymi dla typu "measurements" i podanego identyfikatora w formacie "<type>_<id>_<timestamp>.json". Wczytuje zawartość każdego pliku i parsuje znacznik czasu z nazwy pliku, zwracając wektor par (czas, dane).

Parametry
typeTyp danych ("measurements").
idIdentyfikator czujnika, dla którego wczytywane są dane.
Zwraca
QVector<QPair<QDateTime, QByteArray>> Wektor par zawierających czas i dane.
Nota
Funkcja przetwarza tylko pliki zgodne z wzorcem nazwy dla typu "measurements".

void DataManager::saveHistoricalData ( const QString & type,
const QByteArray & data,
int id = -1 )
static

Zapisuje historyczne dane do pliku JSON.

Zapisuje dane historyczne (np. pomiary, stacje, czujniki) do pliku w katalogu danych aplikacji. Nazwa pliku zależy od typu danych i zawiera znacznik czasu dla pomiarów.

Parametry
typeTyp danych ("stations", "sensors", "measurements").
dataDane do zapisania w formacie QByteArray.
idIdentyfikator, domyślnie -1 (używany dla czujników i pomiarów).

Zapisuje dane historyczne do pliku w katalogu danych aplikacji. Nazwa pliku zależy od typu danych:

  • Dla "stations": "stations.json".
  • Dla "sensors": "<type>_<id>.json".
  • Dla "measurements": "<type>_<id>_<timestamp>.json" z bieżącym znacznikiem czasu. Jeśli katalog nie istnieje, tworzy go.
Parametry
typeTyp danych ("stations", "sensors", "measurements").
dataDane do zapisania w formacie QByteArray.
idIdentyfikator, domyślnie -1 (używany dla czujników i pomiarów).
Nota
Jeśli plik nie może zostać otwarty, dane nie zostaną zapisane.
